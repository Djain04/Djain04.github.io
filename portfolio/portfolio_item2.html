Hangman Game

Hangman is a classic word guessing game for one or more players. One player thinks of a word and the other tries to guess it by suggesting letters or numbers.This project is a simplified version of the game, implemented using C++ programming language to demonstrate the project in the command prompt. The user has 6 chances to guess the word. A message is displayed when the user incorrectly guesses a letter and displays all previously guessed letters so the user doesn't guess them again. Whenever there is an incorrect guess a part of the hangman will be drawn. If the user guesses the correct word within the number of chances, he will be congratulated or else the correct word will be displayed and the user loses. The user has the choice to continue or exit the game.
