Sticky Notes

Sticky Notes become handy when you need to stick to some task! This project is a creative and appealing implementation of sticky notes, developed by using HTML, CSS and Javascript. The user can add a sticky note, rewrite it and delete it. The colors and orientation of the notes is selected randomly to give it more realistic look.
